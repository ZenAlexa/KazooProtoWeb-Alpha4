## 文档组织原则

### README.md (英文)
- **唯一对外文档**
- 面向 GitHub 访客和潜在用户
- 专业、简洁、易读
- 包含完整的安装和使用说明

### docs/ 目录 (中文)
- **内部文档中心**
- 按 Phase 组织技术文档
- 历史文档归档到 `archive/`
- 路线图单独存放 `roadmap/`

### 归档原则
- 测试文件 → `archive/html/`
- 历史文档 → `docs/archive/`
- Phase 文档 → `docs/phaseN/`
- 过时脚本 → 直接删除
